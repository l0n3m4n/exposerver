# Changelog

## [1.3.4] - 2025-11-13

### Fixed
- Fixed a bug where the UI would not load due to incorrect asset handling.
- Fixed a bug where the log file path was not correctly handled.
- Fixed a bug where the `clear_log_file` function would crash.
- Fixed a bug where the script would crash if the `cgi` module was not available.
- Fixed a bug where the `check_and_install_dependencies` function was not cross-platform.

### Changed
- Improved the `check_and_install_dependencies` function to be more robust and cross-platform.
- Refactored the `do_POST` method to be more robust.
- Updated the version of the script to 1.3.4.
